# Roguelike 肉鸽游戏 — 需求与设计文档

## 一、游戏架构说明

本游戏采用 **Canvas 全屏渲染 + HTML HUD 叠加** 架构：

- **Canvas 层**：承载所有游戏实体（玩家、敌人、弹幕、特效）的绘制，尺寸等于视口大小
- **HTML 层**：叠加在 Canvas 上方，负责 HUD（血条、经验条、技能栏）、升级选择面板、死亡界面、Boss 预警条的展示与交互
- **游戏主循环**：`requestAnimationFrame` 驱动，每帧执行 `update → render` 流程
- **数据驱动**：所有常量与数据结构集中于 `constants.js`，游戏逻辑仅消费常量，不硬编码数值
- **组件化**：使用 Vue 3 Composition API，逻辑按功能拆分为独立 composables

## 二、坐标系统设计

### 逻辑坐标 vs 屏幕坐标

| 坐标系   | 说明                                         | 示例                        |
|----------|----------------------------------------------|-----------------------------|
| 逻辑坐标 | 世界空间中的绝对位置，地图逻辑上无限延伸       | 玩家初始 (0, 0)             |
| 屏幕坐标 | Canvas 画布上的像素位置，用于绘制与鼠标输入    | Canvas 中心 = (w/2, h/2)    |

### 转换公式

- **逻辑 → 屏幕**：`screenX = logicalX - cameraX + canvasWidth/2`
- **屏幕 → 逻辑**：`logicalX = screenX + cameraX - canvasWidth/2`
- Y 轴同理
- `toScreen` / `toLogical` 函数兼容 `CanvasRenderingContext2D` 和 `HTMLCanvasElement` 两种参数

### 镜头跟随

镜头始终以玩家为中心平滑跟随：

```
cameraX += (playerX - cameraX) * CAMERA_EASING
cameraY += (playerY - cameraY) * CAMERA_EASING
```

- `CAMERA_EASING = 0.08`：缓动系数，值越小跟随越平滑

## 三、模块划分

| 模块               | 职责                                       |
|--------------------|--------------------------------------------|
| 输入处理           | 键盘（WASD移动 / 技能按键）与鼠标（方向 + 按住持续射击） |
| 玩家系统           | 移动、帧动画、技能释放、受伤闪红、死亡、升级、Buff 效果       |
| 镜头系统           | 缓动跟随玩家，坐标转换                        |
| 敌人系统           | 刷新、AI 行为、帧动画、技能释放、受伤 / 死亡、精英变种     |
| Boss 系统          | Boss 触发检测、阶段转换、多阶段攻击模式、掉落生成     |
| 弹幕系统           | 箭矢 / 追踪弹幕 / 敌人弹幕的飞行与碰撞、穿透攻击逻辑         |
| 特效系统           | 近战劈斩弧、冰冻圈、吸血光环、无敌光环、冲刺残影的渲染与衰减       |
| 事件系统           | 随机事件生成、祭坛 / 神龛 / 宝箱 / 石碑的交互       |

| 升级系统           | 经验积累、等级判断、技能选择面板（无上限升级）  |
| HUD 渲染           | HTML 叠加层：血条、经验条、技能栏、时间、击杀数、Boss 预警条   |
| 渲染管线           | 背景网格 → 事件 → 特效 → 吸血光环 → 敌人 → 弹幕 → 玩家 → HUD |

### 5.2 技能详细说明

#### 🏹 射箭（主动）
- **触发方式**：鼠标按住持续射击
- **效果**：朝鼠标方向发射箭矢，命中敌人造成伤害
- **穿透攻击**（Lv2 解锁）：每升 1 级增加 1 次穿透
- **分裂箭**（Lv7 解锁）：向最近敌人发射分裂箭（伤害 70%）
- **成长**：伤害 ↑、冷却 ↓、弹速 ↑、穿透次数 ↑

#### ⚔️ 近战劈斩（自动触发）
- **触发方式**：范围内有敌人时自动触发
- **效果**：以玩家为中心、朝鼠标方向的扇形范围伤害
- **成长**：伤害 ↑、冷却 ↓、范围 ↑

#### 🔮 追踪弹幕（主动）
- **触发方式**：数字键 3 / 技能栏点击
- **效果**：朝最近敌人发射追踪弹幕，持续修正飞行方向
- **多弹幕**（每级 +1）：最多 5 颗额外弹幕
- **成长**：伤害 ↑、冷却 ↓、范围 ↑、弹速 ↑

#### ❄️ 冰冻定身（主动）
- **触发方式**：数字键 4 / 技能栏点击
- **效果**：范围内敌人定身 + 减速，附带少量伤害
- **成长**：伤害 ↑、冷却 ↓、范围 ↑、持续时间 ↑

#### 🩸 吸血光环（主动）
- **触发方式**：数字键 5 / 技能栏点击
- **效果**：激活持续光环，每秒对范围内敌人造成伤害并回复自身
- **成长**：伤害 ↑、冷却 ↓、范围 ↑、持续时间 ↑、吸血比例 ↑

#### 💪 强身健体（被动）
- **效果**：每级提升最大生命值、移动速度和基础攻击力
- **即时生效**：升级后立即生效，无需重新激活
- **等级生命加成**：每升 1 级额外 +5 最大生命值
- **成长**：最大生命加成 ↑、移速加成 ↑、攻击力加成 ↑

#### 🛡️ 无敌（主动）
- **触发方式**：Q 键 / 技能栏点击
- **效果**：短时间内免疫所有伤害，提升移速与伤害加成
- **视觉特效**：金色光环 + 亮金色边框 + 红色倒计时边 + 最后 2 秒红色闪烁
- **成长**：冷却 ↓、持续时间 ↑、移速加成 ↑、伤害加成 ↑

#### 🔥 魔法阵火雨（主动）
- **触发方式**：R 键 / 技能栏点击
- **效果**：在鼠标位置召唤魔法阵，持续坠落火球并对范围内敌人造成灼烧伤害
- **成长**：伤害 ↑、灼烧伤害 ↑、范围 ↑、火球数量 ↑、持续时间 ↑、冷却 ↓

#### 💨 冲刺（主动）
- **触发方式**：Shift 键
- **效果**：向移动方向快速位移，用于躲避 Boss 和高速敌人
- **无敌帧**：冲刺期间 150ms 无敌
- **残影特效**：冲刺路径留下残影
- **成长**：冷却 ↓、距离 ↑、时长 ↓（更快结束）

## 七、Boss 系统

### 7.1 Boss 触发条件

- **首次触发**：游戏时间 ≥ 120 秒，且玩家等级 ≥ 3
- **后续触发**：上次 Boss 被击败后，冷却 180 秒，且玩家等级 ≥ 上次 Boss 等级 + 2
- **提前预警**：
  - 首次 Boss：提前 30 秒显示 HUD 预警条
  - 后续 Boss：满足条件后立即显示 HUD 预警条

### 7.2 Boss 列表

#### 🔮 暗影聚合体（Lv3 解锁）
- **HP**：300 + 30/级
- **速度**：1.5
- **体型**：3x
- **阶段**：
  - 阶段一（100%~60% HP）：近战拍击、暗影法球
  - 阶段二（60%~25% HP）：+ 分身召唤
  - 阶段三（<25% HP）：+ 狂暴、暗影波
- **掉落**：技能卷轴 ×1、金币袋 ×1

#### 🔥 火焰核心（Lv8 解锁）
- **HP**：600 + 50/级
- **速度**：2.2
- **体型**：2.5x
- **阶段**：
  - 阶段一（100%~0% HP）：火焰柱、火焰弹幕
- **掉落**：技能卷轴 ×2、金币袋 ×2

#### 🔮 虚空织法者（Lv14 解锁）
- **HP**：1000 + 80/级
- **速度**：0（瞬移）
- **体型**：2x
- **阶段**：
  - 阶段一（100%~0% HP）：瞬移、虚空线、减速场、虚空弹幕
- **掉落**：技能卷轴 ×3、金币袋 ×3

### 7.3 Boss 预警系统

- **HUD 预警条**（`BossHudBar.vue`）：
  - 位置：屏幕顶部，Boss 血条下方
  - 显示内容：Boss 名称、倒计时数字、冷却中状态
  - 独立于此 Boss 警告弹窗
- **Boss 警告弹窗**（`BossWarning.vue`）：
  - 显示时机：Boss 即将登场前 2 秒
  - 显示内容：Boss 名称、大号倒计时、最后 2 秒红闪
  - 层级：z-index 200（低于升级面板 250）

## 九、Buff 系统

### 9.1 Buff 类型

| Buff 类型       | 来源         | 效果                           | 是否叠加 |
|-----------------|--------------|--------------------------------|----------|
| `attackBoost`   | 力量祭坛     | 攻击力 +50%                    | 否（刷新时间） |
| `speedShrine`   | 速度神龛     | 移速 +40%、闪避 20%           | 否（刷新时间） |
| `bodyStrength`  | 被动技能     | 最大生命 ↑、移速 ↑、攻击力 ↑   | 是（永久） |

### 9.2 Buff 管理机制

- **施加**：同类型 Buff 刷新持续时间（不叠加）
- **衰减**：每帧递减计时，到期自动移除
- **查询**：`hasBuff(type)` 检查是否有指定类型的 Buff
- **获取倍率**：`getAttackMultiplier()`、`getSpeedMultiplier()`、`getDodgeChance()`

## 十、升级系统

### 10.1 经验获取

- **击杀敌人**：获得敌人 `expReward` 经验值
- **精英敌人**：普通精英 ×1.5、稀有精英 ×2.5
- **Boss**：获得 Boss `xpReward` 经验值

### 10.2 经验等级公式

- **Lv 1~15**：固定表（见 `EXP_LEVEL_TABLE`）
- **Lv 16+**：线性增长
  - Lv15→16 增量 = 3000
  - 每升一级，下一级增量固定 +200
  - 总阈值为 O(n²) 多项式增长

### 10.3 升级流程

1. 经验达到升级阈值 → 等级 +1
2. 立即调用 `recalcPassiveBuffs()`（强身健体效果即时生效）
3. 弹出升级选择面板（3 个选项）
4. 玩家选择后，技能升级或新技能解锁

### 10.4 升级选项生成规则

- **新技能解锁**：尚未拥有的且 `unlockLevel ≤ 当前等级` 的技能
- **已有技能升级**：技能提升 1 级，**无等级上限**（除了冲刺 Lv12 满级）
- **显示信息**：
  - 技能名称、图标
  - 当前等级 / 下一等级
  - 伤害、冷却、范围等关键属性
  - 射箭技能额外显示：穿透次数、分裂箭数量

## 十一、游戏流程

### 11.1 开始游戏

1. 点击 "开始游戏" 按钮
2. 进入游戏主循环
3. 玩家出生在 (0, 0)，初始拥有 **射箭** 技能

### 11.2 游戏进行

1. **移动**：WASD 或方向键
2. **攻击**：鼠标按住持续射箭，其他技能自动触发或手动释放
3. **升级**：经验达到阈值后弹出升级面板，选择技能升级或新技能解锁
4. **事件**：Lv8 后随机生成事件，靠近交互
5. **Boss**：满足条件后触发 Boss 战

### 11.3 游戏结束

1. 玩家 HP ≤ 0 → 显示死亡界面
2. 显示本次游戏数据（击杀数、等级、游戏时长）
3. 点击 "重新开始" 返回开始界面
