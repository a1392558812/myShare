# Roguelike 肉鸽游戏 — 新手指导

欢迎来到 Roguelike 肉鸽游戏！这是一份详细的新手指导，帮助你快速上手游戏。

---

## 📖 游戏简介

这是一款 **肉鸽（Roguelike）射击游戏**，每次游戏都会随机生成技能组合，你需要：

- 🎯 **射击敌人**：用箭矢、魔法、近战攻击消灭敌人
- ⬆️ **升级变强**：击杀敌人获得经验，升级时选择技能强化
- 👹 **挑战 Boss**：击败强大的 Boss，获得丰厚奖励
- 🎲 **随机应变**：每次游戏的技能组合都不同，需要灵活策略

## 🎮 基础操作

### 移动与攻击

| 操作          | 按键                | 说明 |
|--------------|--------------------|------|
| **移动**      | `W` `A` `S` `D` 或方向键 | 控制玩家移动方向 |
| **射箭**      |        鼠标按住        | 朝鼠标方向持续射击 |
| **技能 1**    | `1` 键 / 点击 HUD 图标 | 射箭（默认） |
| **技能 2**    | `2` 键 / 点击 HUD 图标 | 近战劈斩（自动触发） |
| **技能 3**    | `3` 键 / 点击 HUD 图标 | 追踪弹幕 |
| **技能 4**    | `4` 键 / 点击 HUD 图标 | 吸血光环 |
| **冰冻**      | `E` 键 / 点击 HUD 图标 | 冰冻定身 |
| **无敌**      | `Q` 键 / 点击 HUD 图标 | 短时间无敌 |
| **魔法阵**    | `R` 键 / 点击 HUD 图标 | 魔法阵火雨 |
| **冲刺**      | `Shift` 键          | 快速位移（躲避用） |
| **暂停 / 继续** | `空格` 键            | 暂停 / 继续游戏 |
| **自动射箭**  | 点击 HUD 开关按钮     | ActionBar 下方绿色滑块，开启后自动射箭 |

**小贴士**：

- 🖱️ **鼠标瞄准**：所有射击方向都跟随鼠标位置
- ⚔️ **近战自动**：近战劈斩会自动触发，无需手动操作
- 💨 **冲刺保命**：被包围时按 `Shift` 冲刺逃离
- 🏹 **自动射箭**：开启后自动向鼠标位置射箭，省力高效。建议跑图时打开，精确操作时关闭
- ⏸️ **暂停**：按空格键暂停游戏

## 🎯 游戏目标

1. **生存**：避免被敌人击中，保持血量
2. **升级**：击杀敌人获得经验，升级后选择技能强化
3. **挑战 Boss**：击败 Boss 获得大量经验和奖励
4. **变得更强**：每次升级都让你更强大，挑战更高难度

## 📈 升级系统

### 如何升级

1. **击杀敌人** → 获得经验值
2. **经验条满** → 自动升级
3. **弹出面板** → 选择 1 个技能升级或新技能解锁

### 升级选项

每次升级提供 **3 个选项**，包括：

- **新技能解锁**：尚未拥有的技能（如果已达到解锁等级）
- **已有技能升级**：提升已有技能的等级（无上限，除了冲刺 Lv12 满级）

### 推荐升级策略

**新手推荐**：

1. **优先升级射箭**：
   - 伤害 ↑ → 清怪更快
   - 穿透攻击（LvX）→ 一箭穿透多个敌人
   - 分裂箭（LvY）→ 同时攻击多个方向

2. **解锁近战劈斩**：
   - 自动触发，近身自卫
   - 范围伤害，适合被包围时使用

3. **解锁吸血光环**：
   - 持续回血，提高生存能力
   - 适合持久战

**进阶策略**：

- **穿刺流**：主升射箭（穿透 + 分裂箭）+ 追踪弹幕 → 远程清屏
- **近战流**：主升近战劈斩 + 吸血光环 + 无敌 → 近身肉搏
- **均衡流**：射箭 + 近战 + 冰冻 → 远近结合，控制输出

## 🎯 技能详解

### 🏹 射箭（初始技能）

- **特点**：远程攻击，朝鼠标方向射击
- **升级收益**：
  - Lv1~X：伤害 ↑、冷却 ↓、弹速 ↑、**穿透攻击**（LvX 解锁）
  - LvY+：**分裂箭**
- **推荐**：优先升满，主力输出技能

### ⚔️ 近战劈斩（自动触发）

- **特点**：范围内自动攻击，无需手动操作
- **升级收益**：伤害 ↑、冷却 ↓、范围 ↑
- **推荐**：被近身时保命用，适合近战流

### 🔮 追踪弹幕（主动）

- **特点**：自动追踪最近敌人，适合清理漏网之鱼
- **升级收益**：伤害 ↑、冷却 ↓、范围 ↑、弹幕数量 ↑
- **推荐**：穿刺流副技能，清理远程敌人

### ❄️ 冰冻定身（控制）

- **特点**：定身 + 减速敌人，创造输出空间
- **升级收益**：伤害 ↑、冷却 ↓、范围 ↑、持续时间 ↑
- **推荐**：控制流核心，配合射箭使用

### 🩸 吸血光环（生存）

- **特点**：持续回血，提高续航能力
- **升级收益**：伤害 ↑、冷却 ↓、范围 ↑、持续时间 ↑、吸血比例 ↑
- **推荐**：生存流核心，新手必备

### 💪 强身健体（被动）

- **特点**：永久提升属性，无需激活
- **升级收益**：最大生命 ↑、移速 ↑、攻击力 ↑
- **推荐**：每级必升，性价比极高

### 🛡️ 无敌（保命）

- **特点**：短时间无敌，适合紧急避险
- **升级收益**：冷却 ↓、持续时间 ↑、移速加成 ↑、伤害加成 ↑
- **推荐**：Boss 战保命用，进阶必备

### 🔥 魔法阵火雨（AOE）

- **特点**：持续灼烧范围内敌人，适合清屏
- **升级收益**：伤害 ↑、范围 ↑、火球数量 ↑、持续时间 ↑、冷却 ↓
- **推荐**：清屏神器，适合被包围时使用

### 💨 冲刺（位移）

- **特点**：快速位移，150ms 无敌帧
- **升级收益**：冷却 ↓、距离 ↑、时长 ↓（更快结束）
- **推荐**：Boss 战保命 + 走位，进阶必备

## 🔮 Boss 战指南

### 通用技巧

1. **提前准备**：
   - 看到 HUD 预警条 → 预留技能（无敌、冲刺）
   - 血量低 → 优先拾取生命药水

2. **走位要点**：
   - 保持移动，不要停下
   - 利用无敌帧躲避关键攻击
   - 被包围时按 `Shift` 冲刺逃离

3. **技能释放时机**：
   - **无敌**：Boss 释放大招时使用
   - **吸血光环**：血量低时激活
   - **冰冻**：Boss 近身时使用

### Boss 1：暗影聚合体（Lv3 解锁）

- **攻击模式**：
  - 近战拍击：扇形 AoE，提前闪避
  - 暗影法球：追踪弹幕，保持移动
  - 分身召唤（阶段二）：优先清理分身
  - 暗影波（阶段三）：远离 Boss

- **应对策略**：
  - 阶段一：保持距离，远程消耗
  - 阶段二：优先清理分身，避免被围
  - 阶段三：留好无敌，应对狂暴

### Boss 2：火焰核心（Lv8 解锁）

- **攻击模式**：
  - 火焰柱：地面预警，提前离开
  - 火焰弹幕：8 方向弹幕，找空隙躲避

- **应对策略**：
  - 保持移动，不要停下
  - 火焰柱预警时立即离开
  - 弹幕间隙输出

### Boss 3：虚空织法者（Lv14 解锁）

- **攻击模式**：
  - 瞬移：随机传送位置
  - 虚空线：地面红线，持续伤害
  - 减速场：移动速度降低
  - 虚空弹幕：追踪弹幕

- **应对策略**：
  - 最难 Boss，需要高操作
  - 留好无敌和冲刺
  - 优先清理虚空弹幕

## ⚠️ 边界系统

### 软边界机制

地图并非无限大！超出一定范围会受到**渐进式惩罚**：

| 深度       | 伤害（/500ms） | 减速   | 视觉提示                   |
|-----------|---------------|--------|----------------------------|
| 刚越界     | 1% 最大生命   | 10%    | 四边红色 shadow + "危险边界" |
| 深入越界   | 3% 最大生命   | 25%    | 更浓四边红色 + "严重警告"   |
| 致命深度   | 6% 最大生命   | 50%    | 全屏红闪 + "致命边界！返回！"|

### 如何应对

1. **注意警示**：接近边界时屏幕边缘开始出现红晕，上方显示"接近边界"提示
2. **看到红线**：四边红色虚线框是边界线位置，不要跨越
3. **立即返回**：越界后看到警告文字，立刻朝地图中心方向移动
4. **无敌保护**：无敌技能激活时不会受到边界伤害，但仍会减速
5. **安全距离**：从中心向任意方向约 **1.5 个屏幕宽度**内完全安全

## 💡 进阶技巧

### 1. 穿透攻击的正确用法

- **射箭 Lv2** 解锁穿透攻击
- **技巧**：瞄准敌人密集处射击，一箭穿透多个敌人
- **最大化收益**：升级穿透次数

### 2. 分裂箭的释放时机

- **射箭 LvY** 解锁分裂箭
- **技巧**：向敌人密集处射击，分裂箭会自动追踪最近敌人
- **最大化收益**：升级分裂箭数量

### 3. 无敌技能的视觉提示

- **无敌期间**：
  - 脚下金色光环
  - 亮金色边框（默认边）
  - 红色倒计时边（随剩余时间减少）
  - 最后 2 秒红色闪烁
- **技巧**：看到红色闪烁 → 立即撤离，无敌即将结束

### 4. Boss 预警系统的利用

- **HUD 预警条**：提前 30 秒显示，心理准备
- **Boss 警告弹窗**：登场前 2 秒显示，立即规避
- **技巧**：看到预警条 → 预留技能（无敌、冲刺）

### 5. 冲刺的无敌帧利用

- **冲刺期间**：150ms 无敌
- **技巧**：被包围时冲刺，期间无敌 → 逃离
- **进阶**：预判 Boss 攻击，冲刺无敌帧躲避

## ❓ 常见问题

### Q1：为什么射箭穿透攻击不生效？

**A**：需要射箭技能 **Lv2** 才解锁穿透攻击。升级后，箭矢可以穿透多个敌人。

### Q2：如何快速升级？

**A**：

- 击杀敌人获得经验
- 精英敌人经验更多（普通精英 ×1.5、稀有精英 ×2.5）
- Boss 经验丰厚
- 拾取经验书（+200 经验）

### Q3：被 Boss 击败怎么办？

**A**：

- 提前预留无敌技能
- 被包围时按 `Shift` 冲刺逃离
- 升级吸血光环提高续航
- 多练习走位，保持移动

### Q4：技能如何选择？

**A**：

- **新手**：射箭 + 吸血光环 + 近战劈斩
- **进阶**：射箭（满穿透 + 分裂箭）+ 无敌 + 冲刺
- **随意搭配**：每次游戏技能随机，灵活应变

### Q5：如何查看当前技能等级？

**A**：

- 查看 HUD 技能栏，图标右上角显示等级
