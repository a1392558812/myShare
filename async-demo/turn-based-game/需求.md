# 回合制战斗小游戏 - 产品需求文档

## 项目概述

### 项目简介

一款经典单人回合制对战 H5 小游戏，玩家操控己方角色与地图上的 AI 敌方角色轮流出手战斗，包含普攻、技能、血量、状态、胜负判定等核心回合制逻辑，界面简洁复古，适配移动端与 PC 端，所有数据、逻辑均在前端本地运行。

### 核心目标

- 实现标准回合制战斗流程，回合交替、行动判定逻辑完整
- 界面适配 H5 多端，交互流畅，动画反馈清晰
- 纯前端实现，无后端、无网络请求，刷新可重新开局
- 功能模块化，代码易扩展，支持后续新增角色、技能、Buff

## 核心属性系统

### 属性点与属性值区分

- **属性点**：玩家通过升级、自由分配获得的点数
- **属性值**：属性点×系数得出的最终属性数值
- **装备基础属性**：存属性点（需要乘以系数转为属性值）
- **装备基础词条**：存属性点（需要乘以系数转为属性值）
- **装备强力词条**：存属性值（已经乘过系数，直接相加）

### 属性计算公式

| 属性 | 计算公式 |
|------|----------|
| 物理攻击 | 基础物攻值(15) + 玩家物攻点×1.5 + 装备物攻点×1.5 + 装备强力物攻值 |
| 法术攻击 | 基础法攻值(10) + 玩家法攻点×1.1 + 装备法攻点×1.1 + 装备强力法攻值 |
| 防御 | 基础防御值(5) + 玩家防御点×2 + 装备防御点×2 + 装备强力防御值 |
| 速度 | 基础速度值(5) + 玩家速度点×1 + 装备速度点×1 + 装备强力速度值 |
| 生命上限 | 基础生命上限值(100) + 玩家生命上限点×10 + 装备生命上限点×10 + 装备强力生命值 |
| 法力上限 | 基础法力上限值(50) + 玩家法攻点×1.1 + 装备法攻点×1.1 + 装备强力法力值 |

> **注意**：法力上限不依赖独立的属性点，而是通过法攻点自动增加

### 战斗属性（不可加点）

| 属性 | 玩家基础值 | 敌人范围 | 说明 |
|------|-----------|----------|------|
| 暴击率 | 5% | 0% ~ 15% | 实际暴击率受装备加成 |
| 连击率 | 0% | 0% ~ 20% | 实际连击率受装备加成 |
| 最大连击次数 | 1次 | 1 ~ 3次 | 实际值受装备加成 |

#### 暴击计算
- **实际暴击率** = 玩家暴击率 + 所有装备强力词条暴击率（受最大暴击率上限限制）
- **暴击伤害** = 基础伤害 × 暴击倍数(1.5)

#### 连击计算
- **实际连击率** = 玩家连击率 + 所有装备强力词条连击率（受最大连击率上限限制）
- **连击概率衰减**：每次连击后概率衰减至90%
- **连击伤害递减**：每次连击伤害为前次的80%

### 词条生成规则

**基础词条（数量不限制，不重复）**
- 随机数量：0 ~ 全部(6个)
- 词条池：物理攻击、法术攻击、防御、速度、生命、法力

**强力词条（数量受稀有度限制）**
- 随机数量：0 ~ maxAffixes（见上表）
- 词条池包含：强力物理攻击、强力法术攻击、强力防御、强力速度、强力生命、强力法力、暴击率、连击率、连击次数、全属性强化、障碍抗性、忽视障碍异常

### 特殊词条说明

| 词条 | 取值范围 | 专属装备 | 效果 |
|------|----------|----------|------|
| 暴击率 | 1~30 | 武器、手环、腰带 | 增加暴击概率 |
| 连击率 | 1~30 | 武器、手环、腰带 | 增加连击概率 |
| 连击次数 | 1~10 | 项链、玉佩、鞋子 | 增加最大连击次数 |
| 全属性强化 | 1~装备等级 | 护胸、护腿、鞋子、头冠 | 提升所有基础属性 |
| 障碍抗性 | 1~30 | 玉佩、项链、头冠、护胸 | 减少被障碍技能命中概率 |
| 忽视障碍异常 | 1~30 | 武器、腰带、手环 | 增加障碍技能命中概率 |

## 战斗系统

### 回合流程

1. **先手判定**：根据速度属性点判断，速度高者先手
2. **行动执行**：行动方执行操作（普攻/技能/道具/防御/逃跑）
3. **结算**：计算伤害、状态效果
4. **切换**：行动方切换
5. **循环**：重复直到任意一方血量≤0

### 玩家行动

| 行动类型 | 效果 | 消耗 |
|----------|------|------|
| 物理攻击 | 对目标造成物理伤害 | 无 |
| 法术攻击 | 对目标造成法术伤害 | 法力值 |
| 防御 | 本回合减伤50% | 无 |
| 道具 | 使用背包道具恢复生命/法力 | 道具数量 |
| 逃跑 | 退出战斗 | 无 |

### 敌方行动

- **技能使用**：50%概率使用技能（如果有技能可用）
- **道具使用**：血量≤30%时概率使用回血道具
- **技能伤害** = 技能基础伤害 + 敌人魔法攻击 × 0.5

### 障碍技能系统

| 技能 | 成功率 | 持续回合 | 效果 |
|------|--------|----------|------|
| 封印术 | 小概率 | 2回合 | 目标每回合跳过行动 |
| 中毒术 | 概率 | 3回合 | 每回合扣减8%生命+5%法力 |
| 冰冻术 | 大概率 | 2回合 | 免疫伤害，但无法行动 |
| 混乱术 | 小概率 | 2回合 | 随机攻击不分敌我 |

#### 混乱状态规则
- 随机选择目标（玩家、宠物或友方）
- 只能使用物理或法术攻击（50%概率）
- 法术攻击消耗法力，但法力不足时仍可使用

#### 冰冻状态规则
- 无法受到任何伤害
- 无法被施加除冰冻外的任何buff/debuff
- 冰冻术会覆盖其他所有buff
- 无法被治疗或恢复法力

#### 障碍成功率计算
**实际成功率** = 基础成功率 × (1 - 目标障碍抗性/100 + 施法者忽视障碍异常/100)

### Buff/Debuff显示
- 战斗面板显示敌人当前的buff/debuff
- 显示效果名称和剩余回合数
- 同类buff只会显示最新的一个（覆盖机制）

## 地图与敌人系统

### 地图等级系统

- 地图等级可在地图界面设置
- **敌人属性** = 基础属性 × (1 + 1.3 × (地图等级 - 1))
- **技能使用限制**：
  - 地图等级 < 6：敌人不会使用辅助技能
  - 地图等级 < 9：敌人不会使用障碍技能

### 敌人生成

- 所有类型敌人都有概率出现在地图
- 敌人数量：1~4个
- 敌人类型区分物理攻击和法术攻击

## 商店系统

### 购买功能

| 商品类型 | 说明 |
|----------|------|
| 道具 | 生命恢复、法力恢复、百分比恢复 |
| 装备 | 无词条基础装备，等级和稀有度可配置 |

### 售卖功能

- **道具售价** = 购买价格 × 50%
- **装备售价** = 基础价格 × 等级 × 稀有度倍率 × 50%

### 重铸功能

| 重铸类型 | 费用计算 | 效果 |
|----------|----------|------|
| 全刷新 | 50 + 装备等级 × 1 | 刷新所有词条 |
| 单个基础词条 | 30 + 装备等级 × 0.5 | 刷新单个基础词条 |
| 单个强力词条 | 60 + 装备等级 × 0.5 | 刷新单个强力词条 |

### 宠物服务

- **属性重铸**：消耗金币随机改变宠物属性系数
- **技能学习**：消耗金币学习新宠物技能

## 存储系统

- 使用 LocalStorage 本地缓存
- 存储内容：玩家数据、宠物数据、装备背包、道具背包、地图等级

---

## 调试功能

### 调试面板

- **玩家属性修改**：可修改玩家属性点、暴击率、连击率、最大连击
- **宠物属性修改**：可修改宠物属性系数
- **装备生成**：可生成指定稀有度和等级的装备
- **战斗调试**：可在战斗中给单位施加buff
- **战斗重置**：可重置当前战斗

---

## 胜负判定

- **玩家胜利**：所有敌人血量 ≤ 0
- **战斗失败**：玩家和宠物血量 ≤ 0
- **逃跑成功**：玩家主动退出战斗，返回地图

---

## 界面结构

### 非战斗界面
- 地图区域：显示敌人位置
- 角色面板：显示玩家属性、装备、技能
- 背包面板：显示道具和装备
- 商店入口

### 战斗界面
- **上方**：敌方角色（头像、名称、血条、buff显示）
- **下方**：己方角色（头像、名称、血条、法力条）
- **战斗信息区**：当前回合、行动方提示
- **日志区**：战斗操作记录
- **操作按钮区**：普攻、技能、道具、防御、逃跑
- **结算界面**：战斗结果、奖励展示
