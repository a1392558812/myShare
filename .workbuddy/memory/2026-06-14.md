## 实现 obstacle.js 文本特效攻击系统（1/2/3/4键）

- 项目：`async-demo/draw-game-units/`
- 新增文件：`draw-attack/obstacle.js`（~260行）
- 功能：按下 1/2/3/4 键，对 targetList 中所有目标位置从天而降文本特效
  - 1 → "封" 金色 #FFD700（封印）
  - 2 → "毒" 绿色 #66FF33（中毒）
  - 3 → "混" 紫色 #CC44FF（混沌）
  - 4 → "凍" 蓝色 #33CCFF（冰冻）
- 实现要点：
  - 两阶段动画：下落阶段（0.55s 重力飞行）+ 停留阶段（1.5s 脉动发光）
  - 下落阶段：透视缩放(16→38px) + 速度线 + 拖尾粒子 + 光晕三层绘制
  - 停留阶段：脉动(0.9+0.1*sin) + 上下微浮 + 旋转微震 + 4颗环绕光点 + 冲击环(前0.25s)
  - 最后 25% 时间渐隐消失
  - `fireObstacle(key, units, selectedUnit, targetList)` 参数解耦
  - `updateObstacles(ctx, deltaTime)` 封装完整「物理 + 阶段切换 + 绘制」循环
  - attack.vue 导入 `{ updateObstacles, fireObstacle }`，在 onDrawFrame 中调用，在 handleKeyDown 中处理 1/2/3/4 键
