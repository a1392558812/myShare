## 实现 attack.vue 火球从天而降攻击功能

- 文件：`components/attack.vue`
- 功能：按下 `z` 键，`selectedUnit` 向 `targetList` 中所有目标位置各落下一个火球（从天而降）
- 实现要点：
  - `fireballs` ref 数组 + `fireFireball()` 发射函数
  - 火球从 `y = -40`（画布上方）出发，根据 `v0*t + 0.5*g*t² = dist` 计算初速度，约0.7秒到达
  - 多个目标时 X 位置错开 18px 间距避免重叠
  - 重力下落：`fb.vy += fb.g * dt`，`fb.y += fb.vy * dt`（g = 800 px/s²）
  - 到达 `targetY` 后进入爆炸阶段（`explodeTimer = 0.5s`），倒计时结束自动销毁
  - `drawFireball()` 分两阶段绘制：
    - 下落阶段：拖尾火星（径向渐变、生命衰减）+ 三层火球（外层红色光晕 shadowBlur28 → 中层火焰渐变 → 内核白热）+ 顶部5颗喷溅火星
    - 爆炸阶段：初始白光闪爆（前20%）→ 两道扩散冲击波环 → 16颗飞散火星 + 中心余烬
  - 键盘监听在 `handleKeyDown` 中增加 `z` 键分支调用 `fireFireball()`
  - 模板提示更新为 "选择目标，按X射箭 | 按Z火球 | 按住C激光"

## 实现 attack.vue 魔法阵+惊雷攻击功能

- 文件：`components/attack.vue`
- 功能：按下 `v` 键，在鼠标位置创建魔法阵 → 符文升腾 → 惊雷从天劈中法阵 → 余波扩散
- 总动画时长 2.0s，分四阶段：
  - 形成（0→0.5s）：双环渐显 + 顺/逆旋转弧线 + 中心脉动光点
  - 符文升腾（0.2→1.0s）：8个符文从法阵边缘升起，带水平微摆和拖尾光点
  - 惊雷劈下（0.5→1.3s）：三层闪电绘制 + 命中闪爆
  - 余波扩散（1.0→2.0s）：冲击波环 + 12颗地面粒子
- `generateLightning(tx, ty)`：从 y=-30 生成6-9段随机锯齿主路径 + 45%概率分支，精确到达目标
- `drawRune(ctx, rx, ry, size, type)`：4种符文（三角/菱形/圆点/十字星）
- `createMagicCircle()`：读取 `laser.mouseX/Y` 作为法阵位置
- 闪电三层：外层蓝白辉光(10px, shadowBlur20) → 中层青白(4px, shadowBlur14) → 内核纯白(1.8px)
- 键盘监听 `v` 键调用 `createMagicCircle()`
