# 向CSS中注入自定义单位

## 注入单位

```javascript
// 向:root注入--unit: 10px
document.documentElement.style.setProperty("--unit", "10px");

// 验证是否注入成功（可选）
const unitValue = getComputedStyle(document.documentElement).getPropertyValue(
  "--unit",
);
console.log("当前:root的--unit值：", unitValue); // 输出 "10px"
```

## 监听容器变化

### ResizeObserver

```javascript
// 1. 获取需要监听的元素
const targetElement1 = document.getElementById("target1");
const targetElement2 = document.getElementById("target2");

// 2. 创建 ResizeObserver 实例，定义尺寸变化后的回调
const resizeObserver = new ResizeObserver((entries) => {
  // entries 是被监听元素的数组（可同时监听多个）
  for (const entry of entries) {
    // contentBox（内容区）、borderBox（边框 + 内边距 + 内容）、devicePixelContentBox（物理像素）。
    console.log("元素尺寸变化：", entry);
  }
});

// 3. 开始监听目标元素
resizeObserver.observe(targetElement1);
resizeObserver.observe(targetElement2);

// 4. 停止监听（比如组件销毁时，避免内存泄漏）
// resizeObserver.unobserve(targetElement);
// 清空所有监听：resizeObserver.disconnect();
```

### window.resize

```javascript
// 1. 防抖函数（控制触发频率）
function debounce(fn, delay = 200) {
  let timer = null;
  return (...args) => {
    clearTimeout(timer);
    timer = setTimeout(() => fn.apply(this, args), delay);
  };
}

// 2. 获取元素尺寸的函数
function getElementSize(el) {
  const rect = el.getBoundingClientRect(); // 获取元素布局尺寸
  return {
    width: rect.width,
    height: rect.height,
  };
}

// 3. 监听窗口resize
const target = document.getElementById("target");
const handleResize = debounce(() => {
  const { width, height } = getElementSize(target);
  console.log("窗口变化导致元素尺寸：", width, height);
});

// 绑定监听
window.addEventListener("resize", handleResize);

// 初始执行一次
handleResize();

// 移除监听（必要时）
// window.removeEventListener('resize', handleResize);
```

### MutationObserver

```javascript
const target = document.getElementById("target");

// 创建MutationObserver实例
const mutationObserver = new MutationObserver((mutations) => {
  mutations.forEach(() => {
    // DOM/样式变化后，重新获取尺寸
    const { width, height } = target.getBoundingClientRect();
    console.log("DOM/样式变化导致尺寸：", width, height);
  });
});

// 配置监听选项：监听样式变化、子元素变化
mutationObserver.observe(target, {
  attributes: true, // 监听属性变化（比如style、class）
  attributeFilter: ["style", "class"], // 只监听样式/类名属性
  childList: true, // 监听子元素增减
  subtree: true, // 监听子树（子元素的子元素）
});

// 停止监听
// mutationObserver.disconnect();
```

### 注意

- 内存泄漏：所有 Observer 都需在不需要时调用 disconnect()/unobserve() 停止监听（比如 Vue/React 组件的 unmounted/componentWillUnmount 钩子）。
- 防抖 / 节流：window.resize 会高频触发（每秒几十次），必须加防抖，否则会导致页面卡顿。
- 尺寸获取：
  - getBoundingClientRect()：获取元素实际渲染的尺寸（含变换），推荐使用。
  - offsetWidth/offsetHeight：获取布局尺寸（含 border/padding），也可使用。

### 总结

|         方案         |            优点            |          缺点          |           适用场景            |
| :------------------: | :------------------------: | :--------------------: | :---------------------------: |
|    ResizeObserver    | 性能好、精准、专门监听尺寸 |       不兼容 IE        | 现代浏览器、所有元素尺寸监听  |
| window.resize + 防抖 |       兼容所有浏览器       | 仅监听窗口变化、需防抖 | 旧浏览器、元素尺寸随窗口变化  |
|   MutationObserver   |    监听 DOM / 样式变化     | 间接感知尺寸、可能漏判 | 元素尺寸由 DOM / 样式修改导致 |
