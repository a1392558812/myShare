# css 滚动对齐

> 很久没用过这个属性了，这次算是拾遗

|       属性        |                   作用                   |                                  常用值                                   |
| :---------------: | :--------------------------------------: | :-----------------------------------------------------------------------: |
| scroll-snap-type  | 定义容器的滚动对齐类型（作用于滚动容器） | x/y：仅在 X/Y 轴生效；both：双轴生效；mandatory/proximity：对齐严格程度 ① |
| scroll-snap-align |   定义子元素的对齐位置（作用于子元素）   |     start（对齐容器起始端）、center（居中对齐）、end（对齐容器末端）      |
| scroll-snap-stop  |      控制是否允许跳过子元素（可选）      |        normal（允许跳过）、always（强制对齐每个元素，不允许跳过）         |

> ①： 对齐严格程度：mandatory vs proximity
>
> - mandatory：严格对齐，每次滚动必须对齐到子元素的对齐位置（即使滚动距离很短）
> - proximity：接近对齐，只有当子元素滚动到接近对齐位置时，才会自动对齐，否则保持当前位置

```html
<!DOCTYPE html>
<html>
  <head>
    <style>
      /* 滚动容器 */
      .scroll-container {
        width: 300px;
        height: 200px;
        overflow-y: auto;
        border: 2px solid #333;
        /* Y 轴强制对齐 */
        scroll-snap-type: y mandatory;
      }

      /* 滚动项 */
      .scroll-item {
        height: 200px; /* 与容器等高，每次滚动一页 */
        display: flex;
        align-items: center;
        justify-content: center;
        font-size: 30px;
        color: white;
        /* 对齐到容器起始端（顶部） */
        scroll-snap-align: start;
      }

      .carousel {
        width: 500px;
        overflow-x: auto;
        display: flex;
        flex-wrap: nowrap;
        scroll-snap-type: x mandatory; /* X 轴强制对齐 */
        padding: 20px 0;
        border: 2px solid #333;
      }

      .carousel-item {
        flex-shrink: 0;
        width: 500px; /* 与容器等宽 */
        height: 300px;
        display: flex;
        align-items: center;
        justify-content: center;
        font-size: 30px;
        color: white;
        scroll-snap-align: start; /* 对齐容器左端 */
        scroll-snap-stop: always; /* 强制逐个对齐，不允许跳过 */
      }

      .grid-container {
        width: 600px;
        height: 400px;
        overflow: auto;
        scroll-snap-type: both mandatory; /* 双轴对齐 */
        border: 2px solid #333;
      }

      .grid {
        display: grid;
        grid-template-columns: repeat(3, 200px); /* 3列，每列200px */
        grid-template-rows: repeat(4, 200px); /* 4行，每行200px */
      }

      .grid-item {
        scroll-snap-align: start; /* 对齐到网格线 */
        border: 1px solid #fff;
        display: flex;
        align-items: center;
        justify-content: center;
        font-size: 30px;
        color: white;
      }
    </style>
  </head>
  <body>
    <!-- Y 轴滚动对齐 -->
    <div class="scroll-container" style="margin-bottom: 20px;">
      <div class="scroll-item" style="background: #ff4444">1</div>
      <div class="scroll-item" style="background: #00c851">2</div>
      <div class="scroll-item" style="background: #2196f3">3</div>
    </div>

    <!-- 水平滚动对齐 -->

    <div class="carousel" style="margin-bottom: 20px;">
      <div class="carousel-item" style="background: #ff4444">1</div>
      <div class="carousel-item" style="background: #00c851">2</div>
      <div class="carousel-item" style="background: #2196f3">3</div>
    </div>

    <!-- 网格对齐（照片墙之类） -->
    <div class="grid-container" style="margin-bottom: 20px;">
      <div class="grid">
        <div class="grid-item" style="background: #ff4444">1</div>
        <div class="grid-item" style="background: #00c851">2</div>
        <div class="grid-item" style="background: #2196f3">3</div>
        <div class="grid-item" style="background: #f19693">4</div>
        <!-- ... ... -->
      </div>
    </div>
  </body>
</html>
```
